Parkinson’s Disease Smartwatch AI System
Frontend + Backend Integration Information Resource Document
Author: Nitish Patel Project Type: Final Year BE Research & Development Project
1. Project Overview
This project builds an AI‑based Parkinson’s Disease detection and monitoring system using smartwatch
sensor data.
The system combines:
A trained deep learning model (best.pth)
A medical explanation system using Gemini AI
A web‑based frontend interface for user interaction
The model predicts neurological condition categories using smartwatch movement signals.
Target classes:
Healthy Control (HC) Parkinson’s Disease (PD) Differential Diagnosis (DD)
2. System Architecture
The system is divided into four layers:
Frontend (User Interface)
Backend API
AI Model Inference
Gemini Medical Explanation Engine
Architecture flow:
User → Frontend → Backend API → AI Model → Gemini → Result Display

Together they create an interpretable AI assisted diagnostic support system.
